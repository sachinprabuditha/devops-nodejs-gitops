# DevOps Node.js GitOps Project

## Overview
End-to-end DevOps project demonstrating containerization, Kubernetes, GitOps, service mesh, and monitoring using a Node.js application.

## Architecture
Client → Istio Gateway → Node.js Service → Prometheus

## Tech Stack
- Node.js
- Docker
- Kubernetes (Kind)
- GitHub Actions (CI/CD)
- ArgoCD (GitOps)
- Istio (Service Mesh)
- Prometheus (Monitoring)

## Features
- Containerized Node.js app
- Kubernetes deployment and service
- Automated CI/CD pipeline
- GitOps-based deployments with ArgoCD
- Traffic management using Istio
- Metrics exposed and scraped by Prometheus

## How to Run
1. Build and push Docker image
2. Apply Kubernetes manifests
3. Access app via Istio Gateway
4. View metrics in Prometheus

## Skills Demonstrated
DevOps, CI/CD, GitOps, Kubernetes, Observability
